TFTP_Toolbox
============

TFTP_Server:
============

TFTP_Server '/dir/'
Deaemon listens on UDP port 69.
If not specified default dir is /srv/
Server can append missing '/' to dir if need be.

TFTP_Client:
============
TFTP_Client hostname [put|get] filename
* Client still under development *
